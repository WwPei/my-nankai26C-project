# Emoji Dungeon 项目文件功能说明

> 基于：emoji_dungeon_v3 | 总计 58 个源文件（35 .h + 23 .cpp）| 73 个资源文件

---

## 一、项目根目录

| 文件 | 功能 |
|------|------|
| `main.cpp` | **程序入口**。创建 `EmojiDungeonWindow`，设置窗口标题和尺寸，进入 Qt 事件循环。 |
| `CMakeLists.txt` | **CMake 构建脚本**。定义项目名、C++17 标准、3 个子目录、Qt6 模块依赖（Widgets, Multimedia, Svg）、资源文件注册。 |
| `app_icon.rc` | Windows 图标资源脚本，将 `.ico` 注册为应用程序图标。 |
| `README.md` | 项目说明文档。 |

## 二、src/core/ — 配置层（6 文件）

> **被所有其他类型依赖**，定义游戏的全部数据结构和配置表。

| 文件 | 功能 |
|------|------|
| `game_enums.h` | **13 个枚举定义**。包括 `PlayerClassId`（3 职业）、`WeaponId`（3 武器）、`EnemyId`（10 敌人）、`EnemyBehavior`（6 种 AI 行为）、`TraitId`（10 特性）、`UpgradeOptionKind`（3 升级类型）、`BulletStyle`（13 种子弹样式）、`SpecialEffect`（4 种特效）等。 |
| `game_structs.h` | **11 个数据结构体**。定义 `WaveConfig`（波次参数）、`PlayerClassConfig`（职业属性）、`WeaponConfig`（武器属性）、`BulletConfig`（子弹属性）、`EnemyConfig`（敌人 + AI 参数）、`TraitConfig`（14 个特性效果字段）、`UpgradeOption`（升级选项）、`BulletSpecialTemplate`（特殊子弹模板）等。 |
| `game_tables.h` | **所有实例配置表（约 600 行）**。`namespace GameConfig` 中声明了 `kExperienceThresholds`（16 级经验阈值）、`kPlayerClasses`（3 职业）、`kWeapons`（3 武器）、`kEnemies`（10 敌人）、`kTraits`（10 特性）、`kAllUpgradeOptions`（15 个升级选项）等**全部游戏数据** + 对应的 `find*Config()` 查询函数。 |
| `game_data.h` | **聚合头文件**。`#include` 了 `game_enums.h`、`game_structs.h`、`game_tables.h`，其他文件只需包含这一个即可访问全部配置。 |
| `combat_utils.h` | **战斗工具函数声明**。`enemyBaseColor()`、`bulletColor()` 等辅助函数的声明。 |
| `combat_utils.cpp` | **战斗工具函数实现**。按敌人/子弹类型返回对应颜色，用于视图层的底色渲染（无贴图时的 fallback 颜色）。 |

## 三、src/models/interfaces/ — 抽象接口层（6 文件）

> **声明契约，无实现**。所有方法都是 `= 0` 纯虚函数，定义了每种数据对象"能做什么"。

| 文件 | 功能 |
|------|------|
| `player.h` | **玩家抽象接口**。声明 13 个属性 getter（位置/血量/速度/倍率/特性等级），8 个操作 slot（移动/装备/受伤/治疗/冲锋），5 个冲刺方法，5 个信号（移动/血量/武器/击败/冷却）。 |
| `weapon.h` | **武器抽象接口**。声明 4 个属性 getter、14 个操作 slot（瞄准/开火/冷却/弹道/穿透/连击）、7 个查询方法、2 个信号（开火请求/冷却变化）。 |
| `enemy_data.h` | **敌人数据抽象接口**。声明 9 个属性 getter、6 个 AI 方法（行为状态机/冲锋/射击/阶段切换）、7 个信号（位置/血量/击败/射击/自爆/召唤小兵）。 |
| `bullet_data.h` | **子弹数据抽象接口**。声明 11 个属性 getter（位置/方向/伤害/速度/穿透/连击）、10 个操作 slot、2 个信号（位置变化/过期）。 |
| `trait.h` | **特性抽象接口**。声明 3 个属性（id/名称/描述）和 1 个 `applyToPlayer()` 方法。 |
| `game_view.h` | **视图抽象基类**（继承 `QGraphicsObject`）。声明 `syncFromData()` 纯虚同步方法和 `removalRequested()` 信号。是所有 View 的顶层接口。 |

## 四、src/models/basic/ — 模型具体实现层（10 文件）

> **接口的实现者**。每个实现类都持有配置指针 + 运行时状态，全部用 `final` 封闭继承链（除 `BasicBulletData`）。

| 文件 | 功能 |
|------|------|
| `basic_player.h` / `basic_player.cpp` | **玩家具体实现**。214 行逻辑：WASD 边界位移、空格 3 秒 CD 冲刺、10 种特性叠加、属性倍率乘法计算。所有初始值从 `PlayerClassConfig` 读取。 |
| `basic_weapon.h` / `basic_weapon.cpp` | **武器具体实现**。175 行逻辑：每帧冷却递减、CD 归零发射 `fireRequested` 信号、弹道数 = 基础 + 额外弹道、连击计数器、射速缩放。 |
| `basic_enemy_data.h` / `basic_enemy_data.cpp` | **敌人数据具体实现**。339 行逻辑：**5 种 AI 行为状态机**（Chase 追踪 / Charge 冲锋 / ShootAndMove 边走边射 / SuicideBomb 自爆 / Boss 多阶段）、射击冷却计时、冲锋蓄力—冲刺—冷却三段流程、Boss 阶段切换。 |
| `basic_bullet_data.h` / `basic_bullet_data.cpp` | **子弹数据具体实现**。79 行逻辑：按方向 + 速度线性飞行、距离追踪判定过期、穿透计数器。**非 final**，允许 `SpecialBulletData` 继承。 |
| `basic_trait.h` / `basic_trait.cpp` | **特性具体实现**。30 行极简实现：持有 `TraitConfig` 指针，`applyToPlayer()` 委托给 `Player::applyTrait()`。 |

## 六、src/views/ — 视图实现层（10 文件）

> **负责所有可视化渲染**。分为 MV 配对视图（绑定数据模型）和独立 UI 组件。

| 文件 | 功能 |
|------|------|
| `basic_bullet_view.h` / `basic_bullet_view.cpp` | **基础子弹渲染**。98 行绘制逻辑：`QPainter` 绘制彩色圆 + 拖尾残影 + 前向三角箭头。持有 `QPointer<BulletData>` 安全指针。 |
| `basic_enemy_view.h` / `basic_enemy_view.cpp` | **敌人渲染**。168 行绘制逻辑：`drawPixmap()` 渲染 PNG 贴图 + 绿/黄/红血条 + 受击闪 `QTimer` + 命中闪白效果。 |
| `battle_arena_view.h` / `battle_arena_view.cpp` | **战斗场景视图**（继承 `QGraphicsView`）。`drawBackground()` 绘制深色背景 + PNG 背景图片 + 网格线。内存放 `QGraphicsScene`。 |
| `dash_cooldown_widget.h` / `dash_cooldown_widget.cpp` | **冲刺冷却指示器**（独立 `QWidget`）。108 行自定义绘制：灰色底环 + 蓝色扇形进度 + ⚡ 图标 + 秒数文字 + 冷却完毕闪烁动画。 |
| `player_avatar_item.h` / `player_avatar_item.cpp` | **玩家角色图形**（继承 `QGraphicsItem`）。85 行绘制逻辑：渐变色圆形 + **职业对应武器 PNG** 跟随鼠标方向旋转 + **6 种动态面部表情**根据 HP/冲刺/受击切换。 |

## 七、src/pages/ — 页面层（8 文件）

> **用户可见的界面**。`EmojiDungeonWindow` 通过 `QStackedWidget` 管理页面切换。

| 文件 | 功能 |
|------|------|
| `emoji_dungeon_window.h` / `emoji_dungeon_window.cpp` | **窗口导航中枢**（继承 `QMainWindow`）。管理 `QStackedWidget` + 3 个子页面（Start / ClassSelect / GameMain）；`connectNavigation()` 串联所有页面间信号；持有 `GameFactory` 共享工厂、追踪已选特性和当前职业。 |
| `class_select_page.h` / `class_select_page.cpp` | **职业选择页面**。3 张职业卡片（战士/游侠/施法者），点击切换选中态 + 蓝色边框动画，"开始冒险"按钮发射 `classSelected` 信号。 |
| `game_main_page.h` / `game_main_page.cpp` | **核心战斗编排器**（**项目最大文件，约 1500 行**）。持有 3 个子系统指针 + 悬浮 HUD 面板 + 战斗进度条 + Boss 血条 + `handleBattleTick()` 主循环（每 16ms 执行：敌人 AI → 玩家移动 → 武器射击 → 子弹飞行 → 碰撞检测 → 清理 → 波次推进 → 状态刷新）。管理全屏布局 + 升级弹窗 + BGM 切换 + 音效播放。 |
| `upgrade_page.h` / `upgrade_page.cpp` | **升级页面**（**当前已弃用**，由弹窗替代，文件保留）。826 行：5 张选项卡片 + 右侧属性变化预览面板。 |

## 八、src/systems/ — 子系统层（10 文件）

> **封装独立游戏逻辑**。每个 System 不持有数据，通过构造注入接收外部依赖，操作外部容器的引用。

| 文件 | 功能 |
|------|------|
| `game_factory.h` / `game_factory.cpp` | **唯一工厂类**（273 行）。封装对 `game_tables.h` 的所有查询；提供 Data 层和 View 层对象的创建方法；提供复合创建方法返回 `EnemyEntity{Data*, View*}` 结构体。 |
| `wave_manager.h` / `wave_manager.cpp` | **波次管理器**（抽象类 + 匿名实现，502 行）。管理 10 波战斗流程：波次计时、经验累计与等级判定、升级触发与选项生成、Boss 调度表（第 4/7/9 波出 Boss）、出怪数量按波次递增。 |
| `upgrade_resolver.h` / `upgrade_resolver.cpp` | **升级解析器**（212 行）。`applyTrait()` 将特性效果反射到 Player 和 Weapon；`applyWeaponUpgrade()` 应用弹道/穿透/射程/连击等武器强化；追踪 `traitCounts` 等级计数；生成上一次升级的属性变化 HTML 摘要。 |
| `combat_coordinator.h` / `combat_coordinator.cpp` | **战斗协调器**（391 行）。`handleWeaponFireRequested()` 创建子弹（含扇形扩散）；`resolveCombatCollisions()` 碰撞检测（玩家子弹 vs 敌人、敌人子弹 vs 玩家、接触伤害）；`cleanupExpiredBullets()` 和 `cleanupDefeatedEnemies()` 清理。 |
| `enemy_director.h` / `enemy_director.cpp` | **敌人生成器**（384 行）。`spawnTestEnemy()` 按波次随机生成敌人；`spawnBossIfPending()` 三阶段 Boss 管道（含延迟、召唤小兵、阶段切换）；管理 Boss 血条 UI 的显示/更新/隐藏/定位。 |

## 九、src/bullets/ — 特殊子弹系统（5 文件）

> **扩展基础子弹**。Data 侧链式继承（`BasicBulletData → SpecialBulletData`），View 侧并行继承（`BulletView → SpecialBulletView`，与 `BasicBulletView` 同级兄弟）。

| 文件 | 功能 |
|------|------|
| `special_bullet_config.h` | **特殊子弹配置表**。13 个 `BulletSpecialTemplate` 实例定义（普通弹预留 + 12 种特殊子弹：飞刀、太阳法球、月亮法球、哈基米追踪猫、雷霆之矛、回力镖、血之箭、眩晕弹、随机礼物、彗星、击退弹、火箭）；`generateSpecialBulletUpgradeOptions()` 生成升级选项。 |
| `special_bullet_data.h` / `special_bullet_data.cpp` | **特殊子弹数据**（155 行）。继承 `BasicBulletData`，增加 `BulletSpecialConfig`；`advanceSpecialFrame()` 实现 4 种特效：`TrackAndAttach`（追踪挂身持续伤害）、`Stun`（眩晕敌人）、`KnockbackWithBonus`（击退 + 撞墙奖励伤害）。 |
| `special_bullet_view.h` / `special_bullet_view.cpp` | **特殊子弹渲染**（149 行）。继承 `BulletView`（非 `BasicBulletView`！），`paint()` 中 `drawPixmap()` 渲染 emoji PNG 贴图，无贴图时 fallback 金色圆。 |
